    Design Note sobre o diretório /sys.



Sobre o diretório /sys:
======================
    O módulo sys faz parte do executive do kernel base.
Estão aqui as principais rotinas de gerenciamento do sistema.
Os arquivos são:

    abort.c -  Rotinas para abortar o kernel em caso de erro grave.
    channel.c - Gerenciamento de canais de comunicação. Um canal te 2 sockets.
    info.c - Mostra informações sobre o sistema.
    modules.c - Gerencia a utilização de módulos do kernel em kernel mode.
    procedure.c - Está aqui o procedimento de janel do sistema. (procedimento default.).
    server.c - gerencia a utilização de processos servidores, que estão em user mode e em kernel mode.
    services.c - *IMPORTANTE. Aqui está a rotina que atende as systemcalls.
    system.c *IMPORTANTE. Arquivo principal do módulo /sys. Arquivo principal do gerenciador do sistema.
    //...



Sobre desligar o computador:
============================
    Obviamente é função do gerenciador do sistema, desligar o sistema de forma segura e
desligar a máquia também de forma segura.
    @todo: Enquanto o desligamento do hardware não é completamente implementado
           deve-se desligar todod os recursos do sistema para permitir que o usuário
desligue manualmente a maquina de forma segura, sem comprometer o sistema de arquivos ou metafiles.

Via APM:
    This is the basic sequence of APM commands that must be given in order to shut down a computer. For details on exactly how to implement these steps, see the APM article.

+Perform an installation check.
+Disconnect any existing APM interface.
+Connect the Real Mode interface.
+Enable power management for all devices.
+Set the power state for all devices to "Off" (03h).

A summary of ACPI shutdown from the above forum post:
   The ACPI shutdown is technically a really simple thing all that is needed is a outw(PM1a_CNT, SLP_TYPa | SLP_EN ); and the computer is powered off. The problem lies in the gathering of these values especialy since the SLP_TYPa is in the \_S5 object which is in the DSDT and therefore AML encoded.
