## What is Gramado?

Gramado (c) is a 32bit multithreaded operating system.


## Compiling Gramado on Linux.

Linux 4.20.7-1-MANJARO x86_64

gcc (GCC) 8.2.1

GNU ld (GNU Binutils) 2.31.1

NASM version 2.14.02 compiled on Nov 15 2018
